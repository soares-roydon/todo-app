# A full stack todo-app

## Overview

- Users can **sign up** using their name, username, and password
- Users can **sign in** using their username and password
- **Create** a new todo by providing a title and description
- **Edit** an existing todo
- **Delete** an existing todo
- Mark the todo as **done** 

## Tech stack

### Frontend

- React with typescript
- Tailwind

### Backend

- Nodejs
- Express
- Zod
- PostgreSQL
- Prisma
